# Anime Tracker

A full-stack web app to track anime with notes.

## Features
- Add anime
- Track status (plan / watched)
- Edit notes inline
- Delete entries

## Tech Stack
- Next.js
- MongoDB Atlas
- Tailwind CSS

## Setup
1. Clone repo
2. Install dependencies
   npm install
3. Create `.env.local` and add:
   MONGODB_URI=your_connection_string
4. Run:
   npm run dev
